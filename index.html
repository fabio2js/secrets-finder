Secrets Finder Maven Plugin

Single page ufficiale del progetto. Il plugin analizza il repository e segnala secret hardcoded prima che finiscano in produzione.

Vai al repository

Scan ricorsivo

Analizza file e cartelle del progetto in modo automatico.

Pattern predefiniti

Individua API keys, token e password comuni.

Output chiaro

Report leggibile con percorso file e severità del finding.

Quick start

Esegui il plugin con Maven:

mvn com.appshield:secret-finder-maven-plugin:1.0.0-SNAPSHOT:scan
